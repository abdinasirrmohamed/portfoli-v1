# Portfolio v1

A clean and responsive one-page personal portfolio built with HTML, CSS, and a small JavaScript interaction for mobile navigation.

## Features
- Sticky navbar with mobile hamburger menu
- Improved typography using `Space Grotesk` + `Source Sans 3`
- Better vertical rhythm and spacing across sections
- Updated project cards with status labels and refined tag styles
- Smooth scroll between sections
- Sections: Hero, About, Skills, Projects, Contact

## Tech Stack
- HTML5
- CSS3
- Vanilla JavaScript

## Getting Started
1. Clone the repository:
```bash
git clone https://github.com/abdinasirrmohamed/portfolio_v1.git
```
2. Open `index.html` in your browser.
